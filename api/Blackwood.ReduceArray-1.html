Remarks

Like a reductionist's secret weapon, this class simplifies your data into a more manageable form. It reduces the number of points in a series when you want to simplify the data for display or analysis -- or simply not overwhelm the graphing tools.

It uses the Douglas-Peucker algorithm, a recursive approach to reduce the number of points in a series. It finds the points less than given distance from the line segment and removes them.
